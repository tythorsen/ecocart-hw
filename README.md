# Installation and Local Server

## Install Node Modules
```
npm install
```

## Run Vite Dev Server
```
npm run dev

  vite v2.9.10 dev server running at:

  > Local: http://localhost:3000/
  > Network: use `--host` to expose

  ready in 251ms.

```
